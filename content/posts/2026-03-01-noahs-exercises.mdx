I used to draw these worksheets by hand for Noah every morning. Fun ritual, but on some days it was tough to come up with new math problems and creative writing prompts before the coffee kicked in.

Now it's digitised. The template pulls together physical, math, creative writing, and drawing exercises in one place. The core sections run on AI on the server, so each refresh gives Noah a different set of prompts and problems.

Pull this up on an iPad or print it out, and the day's workout is ready.
